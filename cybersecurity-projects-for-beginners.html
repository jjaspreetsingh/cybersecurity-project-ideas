Project 2: Port Scanner

Difficulty: ‚≠ê‚≠ê‚òÜ‚òÜ‚òÜ Time: 3-4 hours Skills: Network security, socket programming, scanning techniques

What You'll Build

A tool that scans a target host for open ports and services.

Why This Project?

Port scanning is a fundamental security assessment technique. You'll understand:

  • TCP/IP networking fundamentals
  • Socket programming
  • Common ports and services
  • Network reconnaissance

Implementation Approach

  1. Use Python's socket library
  2. Implement TCP connect scan
  3. Add port range scanning
  4. Identify common services by port
  5. Add timeout and error handling

Sample Code Structure

import socket
from datetime import datetime

def scan_port(host, port):
    try:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(1)
        result = sock.connect_ex((host, port))
        sock.close()
        return result == 0
    except:
        return False

def scan_host(host, ports):
    open_ports = []
    for port in ports:
        if scan_port(host, port):
            open_ports.append(port)
    return open_ports

Why Build This?

File integrity monitoring is crucial for security. This project teaches:

  • Cryptographic hashing (MD5, SHA-256)
  • File system operations
  • Change detection algorithms
  • Security monitoring concepts

Implementation Strategy

  1. Calculate hash values for monitored files
  2. Store baseline hashes in a database
  3. Periodically re-calculate and compare
  4. Alert on changes

Example Structure

import hashlib
import os
import json

def calculate_file_hash(filepath):
    sha256_hash = hashlib.sha256()
    with open(filepath, "rb") as f:
        for byte_block in iter(lambda: f.read(4096), b""):
            sha256_hash.update(byte_block)
    return sha256_hash.hexdigest()

def monitor_file(filepath, baseline_hash):
    current_hash = calculate_file_hash(filepath)
    if current_hash != baseline_hash:
        return "ALERT: File modified!"
    return "File integrity OK"

Project 5: Encrypted File Vault

Difficulty: ‚≠ê‚≠ê‚≠ê‚òÜ‚òÜ Time: 4-5 hours Skills: Encryption, cryptography, secure storage

What You'll Build

A secure file encryption/decryption system using modern cryptographic algorithms.

Why This Project?

Encryption is fundamental to data security. You'll learn:

  • Symmetric and asymmetric encryption
  • Key management
  • Secure file handling
  • Cryptographic best practices

Implementation Approach

  1. Use cryptography libraries (cryptography, PyCrypto)
  2. Implement AES encryption
  3. Add password-based key derivation (PBKDF2)
  4. Create encrypt/decrypt functions
  5. Add secure key storage

Quick Start

from cryptography.fernet import Fernet
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
import base64

def encrypt_file(filepath, password):
    # Generate key from password
    kdf = PBKDF2HMAC(
        algorithm=hashes.SHA256(),
        length=32,
        salt=b'salt_here',
        iterations=100000,
    )
    key = base64.urlsafe_b64encode(kdf.derive(password.encode()))
    f = Fernet(key)
    
    with open(filepath, 'rb') as file:
        file_data = file.read()
    encrypted_data = f.encrypt(file_data)
    
    with open(filepath + '.encrypted', 'wb') as file:
        file.write(encrypted_data)

Enhancements

  • Add file compression before encryption
  • Implement secure key exchange
  • Add multi-file support

def check_security_headers(url):

        'Strict-Transport-Security': headers.get('Strict-Transport-Security'),
        'Content-Security-Policy': headers.get('Content-Security-Policy')

def test_sql_injection(url):
    payloads = ["' OR '1'='1", "1' UNION SELECT NULL--", "admin'--"]
    for payload in payloads:
        test_url = f"{url}?id={payload}"

Project 9: Firewall Rule Generator

Difficulty: ‚≠ê‚≠ê‚≠ê‚òÜ‚òÜ Time: 4-5 hours Skills: Network security, firewall configuration, access control

What You'll Build

A tool that generates firewall rules based on security policies and network requirements.

Why Build This?

Firewalls are fundamental to network security. This project teaches:

  • Firewall concepts and rules
  • Network access control
  • Rule optimization
  • Security policy implementation

Implementation Strategy

  1. Define security policies (allow/deny rules)
  2. Parse network requirements
  3. Generate iptables or firewall rules
  4. Validate rule syntax
  5. Optimize rule order

Example Structure

def generate_firewall_rules(policy):
    rules = []
    
    # Default deny
    rules.append("iptables -P INPUT DROP")
    rules.append("iptables -P FORWARD DROP")
    rules.append("iptables -P OUTPUT ACCEPT")
    
    # Allow loopback
    rules.append("iptables -A INPUT -i lo -j ACCEPT")
    
    # Allow established connections
    rules.append("iptables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT")
    
    # Apply policy rules
    for rule in policy:
        if rule['action'] == 'allow':
            rules.append(f"iptables -A INPUT -p {rule['protocol']} --dport {rule['port']} -j ACCEPT")
    
    return rules

Advanced Ideas

  • Support multiple firewall types (iptables, ufw, firewalld)
  • Add rule conflict detection
