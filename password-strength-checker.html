Project Overview

In this project, you'll build a comprehensive password strength checker that evaluates passwords based on multiple security criteria. This is your first step into cybersecurity development and will teach you fundamental concepts about password security.

Why This Project Matters

Weak passwords are one of the most common security vulnerabilities. According to cybersecurity reports, over 80% of data breaches involve weak or stolen passwords. By building this tool, you'll learn:

  • How password cracking works
  • Industry standards for password complexity

What You'll Build

Create Password Validation Functions

Start building the core password checking logic. Add this to your password_checker.py file.

import re
import string
import hashlib

def check_length(password):
    """Check if password meets minimum length requirements"""
    if len(password) >= 8:
        return True, "Good length"
    elif len(password) >= 6:
        return False, "Password should be at least 8 characters"
    else:
        return False, "Password too short - minimum 8 characters required"

def check_complexity(password):
    """Check password complexity requirements"""
    has_upper = bool(re.search(r'[A-Z]', password))
    has_lower = bool(re.search(r'[a-z]', password))
    has_digit = bool(re.search(r'\d', password))
    has_special = bool(re.search(r'[!@#$%^&*(),.?":{}|<>]', password))

    criteria_met = sum([has_upper, has_lower, has_digit, has_special])

    if criteria_met == 4:
        return True, "Excellent complexity"
    elif criteria_met >= 3:
        return True, "Good complexity"
    elif criteria_met >= 2:
        return False, "Add more character types for better security"
    else:
        return False, "Password needs uppercase, lowercase, numbers, and special characters"

def check_common_patterns(password):
    """Check for common weak password patterns"""
    common_patterns = [
        r'123456',
        r'password',
        r'qwerty',
        r'abc123',
        r'admin',
        r'letmein'
    ]

    for pattern in common_patterns:
        if pattern.lower() in password.lower():
            return False, f"Contains common pattern: {pattern}"

    return True, "No common patterns detected"

Understanding the code: These functions break down password validation into specific checks, making it easier to understand and maintain.

4

Calculate Password Entropy

Add entropy calculation to measure password strength mathematically.

def calculate_entropy(password):
    """Calculate password entropy (bits of randomness)"""
    char_sets = 0

    if re.search(r'[a-z]', password):
        char_sets += 26
    if re.search(r'[A-Z]', password):
        char_sets += 26
    if re.search(r'\d', password):
        char_sets += 10
    if re.search(r'[!@#$%^&*(),.?":{}|<>]', password):
        char_sets += 32  # Approximate special characters

    if char_sets == 0:
        return 0

    entropy = len(password) * (char_sets ** (1/2)).bit_length()
    return round(entropy, 2)

def get_entropy_rating(entropy):
    """Convert entropy to human-readable rating"""
    if entropy >= 100:
        return "Excellent", "Very strong password"
    elif entropy >= 80:
        return "Good", "Strong password"
    elif entropy >= 60:
        return "Fair", "Moderate strength"
    elif entropy >= 40:
        return "Weak", "Easily crackable"
    else:
        return "Very Weak", "Crackable in seconds"

Why entropy matters: Entropy measures how unpredictable your password is. Higher entropy means stronger security.

Create the Main Analysis Function

Combine all checks into a comprehensive password analysis function.

def analyze_password(password):
    """Comprehensive password analysis"""
    results = {
        'length_check': check_length(password),
        'complexity_check': check_complexity(password),
        'pattern_check': check_common_patterns(password),
        'entropy': calculate_entropy(password),
        'entropy_rating': get_entropy_rating(calculate_entropy(password))
    }

    # Calculate overall score
    score = 0
    if results['length_check'][0]: score += 25
    if results['complexity_check'][0]: score += 25
    if results['pattern_check'][0]: score += 25
    if results['entropy'][0] >= 60: score += 25

    results['overall_score'] = score
    results['recommendations'] = generate_recommendations(results)

    return results

def generate_recommendations(results):
    """Generate improvement recommendations"""
    recommendations = []

    if not results['length_check'][0]:
        recommendations.append("Increase password length to at least 8 characters")

    if not results['complexity_check'][0]:
        recommendations.append("Include uppercase, lowercase, numbers, and special characters")

    if not results['pattern_check'][0]:
        recommendations.append("Avoid common words and patterns")

    if results['entropy_rating'][0] in ['Weak', 'Very Weak']:
        recommendations.append("Use a longer password with more character variety")

    return recommendations

Scoring system: The overall score helps users understand their password's relative strength at a glance.
