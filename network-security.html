A tool that scans a target host for open ports and services.

Why This Project?

Port scanning is a fundamental security assessment technique. You'll understand:

  • TCP/IP networking fundamentals
  • Socket programming
  • Common ports and services
  • Network reconnaissance

Implementation Approach

  1. Use Python's socket library
  2. Implement TCP connect scan
  3. Add port range scanning
  4. Identify common services by port
  5. Add timeout and error handling

Sample Code Structure

import socket
from datetime import datetime

def scan_port(host, port):
    try:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(1)
        result = sock.connect_ex((host, port))
        sock.close()
        return result == 0
    except:
        return False

def scan_host(host, ports):
    open_ports = []
    for port in ports:
        if scan_port(host, port):
            open_ports.append(port)
    return open_ports

Project 9: Firewall Rule Generator

Difficulty: ‚≠ê‚≠ê‚≠ê‚òÜ‚òÜ Time: 4-5 hours Skills: Network security, firewall configuration, access control

What You'll Build

A tool that generates firewall rules based on security policies and network requirements.

Why Build This?

Firewalls are fundamental to network security. This project teaches:

  • Firewall concepts and rules
  • Network access control
  • Rule optimization
  • Security policy implementation

Implementation Strategy

  1. Define security policies (allow/deny rules)
  2. Parse network requirements
  3. Generate iptables or firewall rules
  4. Validate rule syntax
  5. Optimize rule order

Example Structure

def generate_firewall_rules(policy):
    rules = []
    
    # Default deny
    rules.append("iptables -P INPUT DROP")
    rules.append("iptables -P FORWARD DROP")
    rules.append("iptables -P OUTPUT ACCEPT")
    
    # Allow loopback
    rules.append("iptables -A INPUT -i lo -j ACCEPT")
    
    # Allow established connections
    rules.append("iptables -A INPUT -m state --state ESTABLISHED,RELATED -j ACCEPT")
    
    # Apply policy rules
    for rule in policy:
        if rule['action'] == 'allow':
            rules.append(f"iptables -A INPUT -p {rule['protocol']} --dport {rule['port']} -j ACCEPT")
    
    return rules
