Key Libraries

import hashlib
import re
import string

def check_password_strength(password):
    strength = 0
    if len(password) >= 8:
        strength += 1
    if re.search(r'[a-z]', password):
        strength += 1
    if re.search(r'[A-Z]', password):
        strength += 1
    if re.search(r'\d', password):
        strength += 1
    if re.search(r'[!@#$%^&*]', password):
        strength += 1
    return strength

Project 3: File Integrity Monitor

Difficulty: ‚≠ê‚≠ê‚òÜ‚òÜ‚òÜ Time: 3-4 hours Skills: File system security, hashing, monitoring

What You'll Build

A system that monitors files for unauthorized changes using cryptographic hashing.

Why Build This?

File integrity monitoring is crucial for security. This project teaches:

  • Cryptographic hashing (MD5, SHA-256)
  • File system operations
  • Change detection algorithms
  • Security monitoring concepts

Implementation Strategy

  1. Calculate hash values for monitored files
  2. Store baseline hashes in a database
  3. Periodically re-calculate and compare
  4. Alert on changes

Example Structure

import hashlib
import os
import json

def calculate_file_hash(filepath):
    sha256_hash = hashlib.sha256()
    with open(filepath, "rb") as f:
        for byte_block in iter(lambda: f.read(4096), b""):
            sha256_hash.update(byte_block)
    return sha256_hash.hexdigest()

def monitor_file(filepath, baseline_hash):
    current_hash = calculate_file_hash(filepath)
    if current_hash != baseline_hash:
        return "ALERT: File modified!"
    return "File integrity OK"

Advanced Ideas

  • Monitor entire directories recursively

Implementation Approach

  2. Implement AES encryption
  3. Add password-based key derivation (PBKDF2)
  4. Create encrypt/decrypt functions
  5. Add secure key storage

Quick Start

from cryptography.fernet import Fernet
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
import base64

def encrypt_file(filepath, password):
    # Generate key from password
    kdf = PBKDF2HMAC(
        algorithm=hashes.SHA256(),
        length=32,
        salt=b'salt_here',
        iterations=100000,
    )
    key = base64.urlsafe_b64encode(kdf.derive(password.encode()))
    f = Fernet(key)
    
    with open(filepath, 'rb') as file:
        file_data = file.read()
    encrypted_data = f.encrypt(file_data)
    
    with open(filepath + '.encrypted', 'wb') as file:
        file.write(encrypted_data)
